简述搭建过程中 所遇到的坑

  1. yml 语法 严格。在config文件中 “:” 后面必须严格的带上空格

  2. 引入yml 文件的时候 必须在文件名中带上 “”

  3. 所遇到的next 主题 中 部分 功能以及字体 字块 区域 还有头像等信息还需要进一步完善

  4. 在hexo deploy的时候遇到 ERROR ：Deployer is not found的情况 重装 deployer的git命令解决

常用的命令

  1. 新建文章 hexo new [layout] title
  2. 清除缓存 hexo clean
  3. 本地查看调试 hexo g以及 hexo s
  4. 提交组合命令 hexo d -g
